2. Lexical Elements¶

Note

The contents of this section are informational.

2:1 The text of a Hash program consists of tokens.

2.1. Character Set¶

2.1:1 An AsciiCharacter is any Unicode character in the range 0x00 - 0x7F, both inclusive.

Syntax

 LexicalElement ::=
   | Keyword
   | Delimiter

Keyword ::=
     while

Delimiter ::=
     {
   | }
   | [
   | ]
   | (
   | )

2.1:2 Additionally, tokens can be made of identifiers which are composed of AsciiCharacter characters.

2.1:3 Characters are generally considered to be in ascii. Programs can have many AsciiCharacters.
